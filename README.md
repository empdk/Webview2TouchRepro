**Touch Unresponsiveness Reproduction**

*(Reproduced on a Windows Surface Go 3 Tablet)*

1. Launch the app.  
2. Rapidly click several times within the populated product list to trigger the click event.  
3. Observe touch unresponsiveness throughout all elements on the page.  
4. Minimize the application.  
5. Restore the application.  
6. Verify that touch responsiveness is restored.
